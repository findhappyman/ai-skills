# AI Skills

我日常使用的 AI 技能文档，用于 Claude Code / Claude Projects 等 AI 工具。

每个技能是一个独立文件夹，包含 `skill.md`。

## 技能列表

| 文件夹 | 技能名称 | 功能 |
|--------|----------|------|
| [article-optimizer](./article-optimizer/skill.md) | Article Optimizer | 将原始文章优化并适配微信公众号、X（推特）、小红书三个平台，同时生成 AI 配图 Prompt |

## 使用方式

将对应文件夹中的 `skill.md` 内容粘贴到 Claude Projects 的自定义指令中，或作为 Claude Code 的 skill 文件使用。
